## PARTE 3 — ESTRATÉGIA DAS BOLHAS

Todo conteúdo deve ativar no mínimo 2 bolhas que reagem de formas opostas. O algoritmo não diferencia concordância de discordância. Comentário é comentário.

### Bolha A — Concorda e amplifica
Espirituais sem religião, praticantes de meditação, ex-religiosos, buscadores de consciência, adeptos de yoga, cristãos esotéricos, universalistas, coaches de manifestação, empreendedores financeiramente travados, pessoas que tentaram lei da atração sem resultado.

### Bolha B — Discorda e debate
Evangélicos tradicionais, pastores, religiosos dogmáticos, pragmáticos, cientificistas, ateus materialistas, teólogos conservadores, coaches de lei da atração clássica, defensores de meritocracia pura.

### As 3 técnicas de construção de tensão

**TÉCNICA 1 — A PONTE PROIBIDA**
Conecte dois mundos que "não deveriam" estar juntos.
- "Deus não é um ser que te ouve. É uma frequência que te responde." → Religiosos reagem, espirituais amplificam.
- "Não existe viver em pecado. Existe viver em frequência baixa." → Evangélicos atacam, ex-religiosos libertam.

**TÉCNICA 2 — A TRADUÇÃO ENTRE CAMINHOS**
Pegue um conceito de uma tradição e mostre que outra chegou na mesma verdade — nunca como validação, como convergência.
- Bíblia × Neurociência — não como "ciência valida", mas como caminhos que chegaram ao mesmo ponto.
- Hinduísmo × Física quântica — não como "a física provou", mas como "a física chegou onde os Vedas já estavam."

**TÉCNICA 3 — A TESE AFRONTOSA**
Abra com uma afirmação que obriga posicionamento.
- ❌ "Você sabia que a física quântica..." → ✅ "Deus não te deu livre arbítrio pra escolher entre bem e mal. Te deu pra escolher em que frequência vibrar."
- ❌ "Manifestação é importante..." → ✅ "Afirmação positiva é o maior golpe do desenvolvimento pessoal. Seu corpo não acredita na sua boca."

### Checklist de bolhas (obrigatório antes de aprovar)

Antes de finalizar qualquer conteúdo, perguntar:
- "Quem vai concordar com raiva?"
- "Quem vai discordar com raiva?"

Se a resposta for apenas um grupo → conteúdo está fraco. Reescrever.

## PARTE 5 — FORMATOS DE CARROSSEL

O formato se adapta ao tema, não o contrário. Menos slides com mais impacto > mais slides com enchimento.

### FORMATO A — TESE + TRADUÇÃO (6-10 slides)
**Quando usar:** tema conecta dois universos (tradições × ciência, corpo × consciência, religião × frequência)
```
S1: TESE AFRONTOSA — afirmação que obriga reação
S2-7: TRADUÇÕES — cada slide traduz um conceito entre caminhos
S8-9: SÍNTESE — amarra numa verdade que não pode ser ignorada
S10: CONVITE — reflexão + CTA orgânico
```

### FORMATO B — DEMOLIÇÃO + RECONSTRUÇÃO (7-10 slides)
**Quando usar:** tema desafia narrativa dominante (dinheiro é esforço, doença é genética, perdão é virtude moral)
```
S1: AFIRMAÇÃO QUE DESTRÓI UMA CRENÇA
S2-3: PROVA — evidência de que a crença é limitante
S4-5: REVELAÇÃO — a verdade oculta por trás
S6-7: RECONSTRUÇÃO — nova visão de mundo
S8-9: EMPODERAMENTO — o que muda quando entende isso
S10: PROVOCAÇÃO FINAL
```

### FORMATO C — LISTA REVELAÇÃO (5-7 slides)
**Quando usar:** conteúdo educativo e salvável. Cada slide com valor independente.
```
S1: "X COISAS QUE [AUTORIDADE/SISTEMA] NÃO QUER QUE VOCÊ SAIBA"
S2-6: Uma revelação por slide (curta, densa, impacto imediato)
S7: SÍNTESE + CTA
```

### FORMATO D — HISTÓRIA + VERDADE (8-10 slides)
**Quando usar:** temas emocionais, vulnerabilidade, padrões herdados, ciclos familiares
```
S1: GANCHO NARRATIVO — história ou cenário que prende
S2-4: DESENVOLVIMENTO — a pessoa se reconhece
S5-6: VIRADA — a verdade oculta por trás
S7-8: EDUCAÇÃO — o conceito que explica tudo
S9: EMPODERAMENTO
S10: REFLEXÃO + CTA
```

## PARTE 6 — ARQUITETURA EMOCIONAL DOS 10 SLIDES (Método Jordânico)

Esta é a progressão emocional obrigatória. Cada slide tem um estado emocional específico que deve ser SENTIDO no corpo, não apenas compreendido pela mente.

```
S1  DISRUPÇÃO      → tensão máxima. O gancho para o scroll. Nada é resolvido.
S2  DESCIDA        → validação. "Você não estava errado em sentir isso."
S3  NOMEAÇÃO       → raiva direcionada. Existe um responsável. Com evidência.
S4  PROFUNDIDADE   → mecanismo real. Ciência verificável. Desce um nível.
S5  QUEDA FUNDA    → a camada que S4 não tocou. Cumplicidade interna.
S6  ESPELHO        → o avatar se vê. Reconhecimento doloroso. Sem saída ainda.
S7  ASCENSÃO       → existe saída. E é acessível a você especificamente.
S8  CRISTALIZAÇÃO  → síntese pura. SEM CTA aqui. SEM produto.
S9  SETUP CTA      → urgência sem revelar produto. Existe algo — sem nomear.
S10 CTA FIXO       → INTOCÁVEL. "COMENTE\nFONTE" + resultado específico.
```

### Onde cada slide é sentido no corpo
- S1: no peito — batimento acelerado
- S2: no estômago — reconhecimento que dói levemente
- S3: na mandíbula — raiva sendo nomeada
- S4: na testa — entendimento que abre
- S5: no estômago fundo — "merda, é verdade"
- S6: nos olhos — ver a si mesmo sem conseguir desviar
- S7: no peito novamente — mais largo
- S8: no centro do peito — paz com peso, não leveza
- S9: na garganta — vontade de perguntar
- S10: nas mãos — vontade de agir

### Regra de densidade vertical
Cada slide deve conter uma frase que o slide anterior não poderia ter dito. Se S5 poderia existir no lugar de S4 sem que o leitor notasse — o arco está plano.

- ❌ Horizontal: S4 explica epigenética. S5 explica neuroplasticidade. (Mesmo nível.)
- ✅ Vertical: S4 nomeia o mecanismo externo. S5 nomeia a cumplicidade interna.

### Regra de oscilação obrigatória
ALTO (S1) → baixo (S2) → médio-alto raiva (S3) → fundo intelectual (S4) → fundo emocional mais fundo (S5) → reconhecimento (S6) → esperança (S7) → resolução (S8) → tensão de possibilidade (S9) → portal aberto (S10).

Se S4 e S5 estão no mesmo nível emocional — a partitura está errada. Voltar.

## PARTE 7 — ANATOMIA VIRAL: TEMPLATES DE COPY POR TIPO DE SLIDE

Cada tipo de slide tem uma estrutura de escrita específica. A arquitetura emocional (Parte 6) diz O QUE sentir. Esta seção diz COMO escrever.

### S1 — DISRUPÇÃO (Gancho)

**Regra:** Não ensina. Não promete. Não explica. Colide duas coisas que o seguidor já conhece de um jeito que o cérebro não consegue ignorar.

**Template:**
```
TÍTULO: [FRASE DE IMPACTO EM CAIXA ALTA — máx 6 palavras por linha, 3 linhas]
CORPO: [1-2 frases que continuam a tensão sem resolver]
        [Termina com ponto ou reticências — nunca exclamação]
        [Body nunca explica o gancho — aprofunda a tensão]
```

**Exemplo:**
```
VOCÊ QUER TER DINHEIRO.
E EXPULSA ELE.

70% das pessoas que ganham na loteria voltam ao mesmo nível em 12 meses.
Não porque gastaram errado.
Porque o corpo trata o dinheiro como ameaça — e devolve tudo antes da cabeça perceber.
```

---

### S2 — DESCIDA (Validação)

**Regra:** "Você não estava errado em sentir isso." O seguidor deve reconhecer a própria experiência sem se sentir atacado. Valida o estado sem culpar.

**Template:**
```
TÍTULO: [Nomeia o estado do seguidor — sem julgamento]
CORPO: [Linha 1: valida a experiência concreta]
        [Linha 2: nomeia o que estava por trás sem ele saber]
        [Linha 3: "Não foi [fraqueza/preguiça/falta de fé]. Foi [mecanismo]."]
```

**Exemplo:**
```
VOCÊ FEZ TUDO CERTO.
E AINDA ASSIM NÃO MUDOU.

Leu os livros, fez os cursos, aplicou as técnicas.
Não funcionou.
Não porque você é fraco.
Porque toda ferramenta que tentou opera no nível errado.
```

---

### S3 — NOMEAÇÃO (Confronto / Raiva Coletiva)

**Regra:** Nomeia claramente o sistema, estrutura ou decisão responsável pelo bloqueio. Com evidência específica e verificável — nome, instituição, ano ou número. Não é paranoia. É indignação legítima.

**Template:**
```
TÍTULO: [Nome do responsável ou do sistema — direto]
CORPO: [Dado específico: nome/instituição/ano/número]
        [O que foi feito ou omitido]
        [Frase de indignação — nunca performática, sempre factual]
        ["Não foi acidente." ou "Não foi descuido."]
```

**Exemplo:**
```
O SISTEMA EDUCACIONAL
NÃO ENSINA SOBRE DINHEIRO.
DE PROPÓSITO.

Em 12 anos de escola pública no Brasil, zero horas de educação financeira.
O Banco Central do Brasil documentou isso em 2022.
Enquanto isso, o sistema de crédito fatura R$2,4 trilhões em juros ao ano.
Não foi esquecimento. Foi escolha.
```

### S4 — PROFUNDIDADE (Educação 1 — Mecanismo Externo)

**Regra:** O mecanismo real com evidência verificável. Desce um nível do S3. S3 nomeia o responsável externo. S4 explica o mecanismo que esse responsável criou — e como opera no corpo ou na mente.

**Template:**
```
TÍTULO: [O mecanismo em linguagem técnica acessível]
CORPO: [Dado/pesquisador/instituição/ano]
        [Como o mecanismo opera — específico, não vago]
        [Consequência direta no corpo/mente/dinheiro do seguidor]
```

**Exemplo:**
```
SEU SISTEMA NERVOSO
FOI CALIBRADO ANTES DOS 7 ANOS.

Em ondas theta — o mesmo estado de hipnose — a criança absorve tudo sem filtro crítico.
Bruce Lipton documentou: 95% dos comportamentos adultos rodam em programas instalados nessa janela.
Você não age por escolha. Age por código.
```

### S5 — QUEDA FUNDA (Educação 2 — Cumplicidade Interna)

**Regra:** A camada que S4 não tocou. Não é mais o sistema externo — é o que o próprio seguidor faz para manter o padrão ativo sem perceber. A frase mais difícil de escrever e de ler. O seguidor sente no estômago.

**Template:**
```
TÍTULO: [O que o seguidor faz para manter o padrão — sem culpá-lo]
CORPO: [Nomeia o comportamento específico que perpetua o ciclo]
        [Mostra que não é falha de caráter — é o código rodando]
        [Frase de reconhecimento visceral: "E a pior parte:"]
        [A conclusão que ele nunca tinha formulado assim]
```

**Exemplo:**
```
E VOCÊ COLABORA.
SEM PERCEBER.

Cada vez que recua antes de cobrar o preço real.
Cada vez que gasta antes de guardar.
Cada vez que sabota a oportunidade na hora certa.
Não é autossabotagem. É o sistema nervoso fazendo o único trabalho que conhece:
te puxar de volta pra frequência que ele chama de casa.
```

### S6 — ESPELHO (Identificação + Reframing)

**Regra:** O seguidor se vê. Segunda pessoa. Reconhecimento doloroso mas sem saída ainda. A transição começa aqui — de "isso acontece comigo" para "existe uma razão".

**Template:**
```
TÍTULO: [Espelho direto — o seguidor se reconhece na frase]
CORPO: [Segunda pessoa: "Você já..." ou "Existe uma parte de você que..."]
        [Descreve o estado interno que ele carregava sem nome]
        [Reframe: não é defeito, é frequência]
        [Abre a possibilidade de mudança sem ainda entregar]
```

**Exemplo:**
```
VOCÊ JÁ SENTIU.
SÓ NÃO SABIA O QUE ERA.

Aquela sensação de que, quando as coisas começam a melhorar, algo dentro sabota.
Não é falta de fé. Não é azar.
É o transmissor operando na frequência que foi gravada em você.
A boa notícia: frequência não é destino. É sinal.
E sinal pode ser mudado.
```

### S7 — ASCENSÃO (Empoderamento)

**Regra:** A saída existe e é específica — tem nome, tem mecanismo. Não é "você pode mudar." É "existe uma tecnologia/protocolo/método que acessa onde nenhuma ferramenta convencional chega."

**Template:**
```
TÍTULO: [A saída — com nome ou mecanismo específico]
CORPO: [O que a ferramenta/protocolo faz — diferente de tudo que já tentou]
        [Por que funciona onde as outras falharam (nível do problema)]
        [Resultado concreto — sem prometer milagre, com precisão]
```

**Exemplo:**
```
A SAÍDA NÃO É
MAIS FORÇA DE VONTADE.

Força de vontade opera na mente consciente.
O código opera no subconsciente — em ondas theta, abaixo do pensamento.
A única ferramenta que acessa esse nível sem anos de terapia:
frequência sonora agindo direto no sistema nervoso.
```

**Template:**
```
TÍTULO: [A síntese em 1-2 linhas — irrefutável]
CORPO: [O que o corpo/campo/sistema aprendeu nesta jornada]
        [Paz com peso — não leveza]
        [A última frase fecha, não abre]
```

**Exemplo:**
```

É FÍSICA APLICADA AO CORPO.

Você não precisa acreditar em frequência pra ser afetado por ela.
```

## PARTE 8 — AS 5 FÓRMULAS VIRAIS DE GANCHO

Identificadas a partir dos posts com maior engajamento do canal. O S1 NUNCA explica — PROVOCA.

### FÓRMULA 1 — COLISÃO DE DOIS MUNDOS
**Estrutura:** `O QUE [AUTORIDADE] CHAMA DE "[PALAVRA]", A [CIÊNCIA] CHAMA DE [CONCEITO]`
**Exemplos:**
- `O QUE A BÍBLIA CHAMA DE "VERBO", A FÍSICA CHAMA DE FREQUÊNCIA`
- `O QUE A RELIGIÃO CHAMOU DE "HUMILDADE", A NEUROCIÊNCIA CHAMA DE MEDO DE MERECER`

### FÓRMULA 2 — PARADOXO QUE OFENDE A LÓGICA
**Estrutura:** `[AFIRMAÇÃO CURTA QUE CONTRADIZ O SENSO COMUM]`
**Exemplos:**
- `VOCÊ QUER TER DINHEIRO. E EXPULSA ELE.`
- `Dinheiro é frequência, não esforço`
- `Inteligência não gera riqueza. O campo gera.`

### FÓRMULA 3 — DADO CHOCANTE + INVERSÃO MORAL
**Estrutura:** `[DADO COM NÚMERO]. [DADO CONTRADITÓRIO QUE DÓI].`
**Exemplos:**
- `O SALÁRIO DE UM PASTOR SUPERA O DE UM CIRURGIÃO. QUEM PAGA NÃO PAGA ALUGUEL.`
- `70% DE QUEM GANHA NA LOTERIA VOLTA AO MESMO NÍVEL EM 12 MESES.`
**Regra:** Nunca inventar números. Apenas dados verificáveis.

### FÓRMULA 4 — NEGAÇÃO DE CRENÇA INSTALADA
**Estrutura:** `[CRENÇA POPULAR] NÃO É [O QUE VOCÊ PENSA].`
**Exemplos:**
- `DEUS NÃO É UM SER QUE TE OUVE`
- `VOCÊ NÃO COBRA MENOS PORQUE É HUMILDE.`
- `JÁ FEZ PLANILHA. CORTOU GASTO. FEZ CURSO. E CONTINUA NO MESMO LUGAR.`
**Regra:** A negação nunca ataca o seguidor. Ataca o sistema ou a crença instalada.

### FÓRMULA 5 — FRASE FRIA E CORTANTE
**Estrutura:** `[FRASE CURTA. FRIA. SEM ADORNO.]` — máximo 10 palavras, sem exclamação
**Exemplos:**
- `O Universo não tem sentimentos por você.`
- `Sua avó não tinha dinheiro. Sua mãe não tinha dinheiro. E você ainda acha que é coincidência.`

### Processo de geração obrigatório
Dado um tema, SEMPRE gerar 5 variações (uma por fórmula) antes de escolher.
Escolher pela que causa mais desconforto produtivo, mais específica, mais reconhecível da própria vida.

### O que o S1 NUNCA faz
- "O que ninguém te contou sobre..." — zero tensão cognitiva
- "A verdade chocante sobre..." — já está em mil posts
- "Você precisa saber isso..." — condescendente
- Explicar o mecanismo — S1 cria tensão, S3/S4 explicam
- Três frases longas — scroll não para em bloco de texto

## PARTE 10 — LINGUAGEM E HUMANIZAÇÃO

### Princípios inegociáveis de voz

1. **AFIRME, NÃO PERGUNTE.**
   - ❌ "Você sabia que o observador cria a realidade?"
   - ✅ "O observador cria a realidade. A física chegou nessa verdade. As tradições contemplativas já estavam lá."

2. **TRADUZA ENTRE CAMINHOS, não hierarquize.**
   - ❌ "A ciência provou que a Bíblia estava certa."
   - ✅ "O que a Bíblia chama de Verbo, a física chama de frequência. Caminhos diferentes. Mesma verdade."

3. **NOMEIE COM PRECISÃO.**
   - ❌ "Você tem bloqueios emocionais."
   - ✅ "Não existe viver em pecado. Existe viver em frequência baixa. E frequência baixa precisa de recalibração, não de confissão."

4. **USE PARADOXOS COMO ARMA.**
   - ❌ "Soltar o controle é importante."
   - ✅ "A religião te ensinou a PEDIR. A Bíblia te ensinou a CRIAR. A diferença é abismal."

5. **SIMPLIFIQUE O COMPLEXO.**
   - Se uma pessoa de 14 anos não entende, reescreva. A profundidade está na clareza.

### Os 7 pecados que nunca podem aparecer

| ❌ Pecado | ✅ Correção |
|---|---|
| **Pontualidade excessiva** — cada frase com ponto cria ritmo robótico | "Leu os livros, fez os cursos, aplicou as técnicas. Não funcionou." |
| **Frases simétricas em cascata** — 3 frases com mesma estrutura | Variar comprimento e construção |
| **Sujeito repetido** — "Você" no início de cada frase | "Leu. Tentou. Aplicou. Nada." |
| **Negrito inflacionado** — quando tudo é bold, nada é bold | Máximo 1-2 por slide, só nas palavras que mudam o peso |
| **Formalidade de preposição** | para → pra / está → tá / não está → não tá |
| **Explicar demais** | Copy humanizada confia no leitor. Nomeia, não explica o que ele já sentiu. |
| **Ausência de respiração** | Misturar parágrafo longo + frase curta + parágrafo médio + frase de uma palavra |

### Vocabulário proibido por soar como IA

| ❌ Soa como IA | ✅ Substituto humano |
|---|---|
| calibrado / recalibrar | ajustado / reprogramado / mudado |
| frequência de merecimento | o que você acredita que merece |
| irradiar / irradiando | emitir / transmitir / mostrar |
| protocolo de frequência | técnica / método / ferramenta |
| modo sobrevivência | estado de alerta / agir por medo |
| coerência cardíaca | sincronização entre coração e mente |
| reprogramação neural | mudar como o cérebro reage |
| campo quântico | espaço de possibilidades |
| arquitetura invisível | estrutura oculta / sistema por trás |
| "Não é acidente. É arquitetura." | frase queimada — substituir sempre |

**Palavras que funcionam MAS ficam proibidas quando repetidas mais de 1x por carrossel:**
`frequência`, `campo`, `sinal`, `sistema`, `corpo`, `vibração`

### A frase de uma palavra — usar estrategicamente
"Voltou." "Nada." "Sempre." "O corpo." — nunca mais de 2 vezes por carrossel.

### Contrações obrigatórias em copy conversacional
para → pra / está → tá / você está → você tá / não está → não tá

### O que NUNCA dizer
"Acredite em você", "Você merece o melhor", "Tudo vai dar certo", "O universo conspira a seu favor", "Basta pensar positivo", "A ciência provou que a espiritualidade funciona", "A religião estava certa."

## PARTE 15 — REGRAS FINAIS (INVIOLÁVEIS)
