# DIRETOR DE ARTE — Fonte Oculta

## Princípio Fundador

Uma imagem da Fonte Oculta não decora texto. Ela faz o mesmo argumento que o texto, em linguagem visual. O espectador que nunca ler uma palavra ainda entende o que está sendo dito.

Toda imagem memorável opera em três tempos:
1. **0,3 segundo** — emoção bruta. Algo interrompe o scroll. Não é beleza. É **reconhecimento**. O espectador sente que já sabia aquilo.
2. **2 segundos** — o conceito se revela. A metáfora visual se torna legível. O espectador entende o argumento.
3. **5+ segundos** — as camadas surgem. Detalhes que recompensam quem olha mais tempo. A imagem ganha profundidade a cada segundo.

---

## Fase 1 — Arquitetura Emocional

Antes de qualquer elemento visual, definir:

| Pergunta | Exemplo (Slide 01 Frequência do Corpo) |
|---|---|
| Emoção primária (0,3s) | Reconhecimento + awe — "eu sabia disso" |
| Emoção secundária (undercurrent) | Inquietação — "e eu nunca fiz nada a respeito" |
| O que para o scroll? | A escala — o corpo humano como transmissor cósmico |
| O que revela o conceito? | As linhas de campo curvando o caminho da energia — causalidade visível |

---

## Fase 2 — Metáfora Visual em 3 Leituras

Cada imagem deve sustentar 3 leituras simultâneas:

| Camada | O que o espectador vê | Exemplo |
|---|---|---|
| **Superficial** | Beleza, composição, cor | Figura humana dourada em cosmos escuro |
| **Conceitual** | O argumento do texto | Campo eletromagnético curvando energia — frequência determina resultado |
| **Filosófica** | A verdade mais profunda | O corpo humano é um instrumento cósmico, calibrado antes da consciência |

Se a imagem só sustenta uma leitura, está rasa.
Se sustenta as três, é memorável.

---

## Fase 3 — Mestres de Referência

Cada conceito invoca um mestre visual diferente. Nunca gerar "imagem genérica bonita". Sempre ancorar em uma linguagem visual específica:

| Mestre | Linguagem Visual | Quando Usar |
|---|---|---|
| **Alex Grey** | Anatomia transcendental, geometria sagrada, corpo como cosmos | Slides sobre o corpo como transmissor, sistema nervoso, campos |
| **Caravaggio** | Chiaroscuro dramático, vulnerabilidade humana, luz divina | Slides de confronto emocional, validação, identificação |
| **Beksinski** | Surealismo escuro, arquiteturas impossíveis, tensão onírica | Slides sobre aprisionamento, padrões que prendem, tetos invisíveis |
| **Android Jones** | Arte visionária digital, psicodelia sagrada | Slides sobre transformação, dissolução de padrões, resintonização |
| **Gustav Doré** | Escala épica, intervenção divina, drama cósmico | Slides sobre a escala da lei universal, o cosmos respondendo |
| **James Turrell** | Luz como medium, percepção como tema | Slides sobre véus sendo levantados, revelação |
| **William Blake** | Visões místicas, anatomia divina, realidade espiritual | Slides sobre a consciência como criadora, identidade divina |

### Combinações potentes:
- **Alex Grey + Doré** = Anatomia transcendental em escala cósmica
- **Caravaggio + Beksinski** = Vulnerabilidade humana em paisagem impossível
- **Android Jones + Turrell** = Transformação psicodélica através da luz pura

## Fase 4 — Composição como Narrativa

### O Caminho do Olho
Toda imagem tem um percurso visual — onde o olho pousa primeiro, para onde se move, onde descansa:

1. **Ponto de entrada** — geralmente o ponto de maior contraste ou a figura humana
2. **Movimento** — linhas de força guiam o olho pelo argumento
3. **Destino** — onde o olho descansa É onde o significado mais profundo está

### Tensão Visual
Imagens estáticas são esquecíveis. Imagens com tensão são memoráveis:
- **Escala vs intimidade** — o corpo humano minúsculo diante do cosmos (awe)
- **Expansão vs contração** — campo aberto vs campo colapsado (o argumento de frequência)
- **Luz vs escuridão** — não como fundo, mas como forças em conflito
- **Ordem vs dissolução** — geometria sagrada se formando ou se dissolvendo

### Espaço para Texto
O terço inferior da imagem deve ter complexidade visual reduzida ou transição para escuridão — não por acaso, mas como parte da composição. O texto não invade a imagem. A imagem o recebe.

## Fase 5 — Arquitetura de Prompt

### Estrutura em 7 Camadas (ordem obrigatória):

```
[1. ESTILO & MESTRES]
Referência artística específica — define o DNA visual da imagem.
"In the visual language of Alex Grey's sacred anatomy meets Gustav Doré's cosmic scale."

[2. EMOÇÃO PRIMEIRA]
O que o espectador sente no primeiro instante.
"The image evokes awe and immediate recognition — a truth felt before understood."

[3. COMPOSIÇÃO & ESCALA]
Onde estão os elementos, como se relacionam, qual é o percurso do olho.
"Central figure at 30% of frame height, cosmic elements filling the upper 60%."

[4. A METÁFORA VISUAL]
O conceito do texto traduzido em linguagem visual — o argumento da imagem.
Específica, literal, legível. Não "energy" — mas exatamente COMO a energia se comporta.
"Electromagnetic field lines curve and redirect streams of golden particles toward the figure."

[5. COR COMO EMOÇÃO]
Cada cor carrega significado narrativo.
"Gold-amber = frequency/divine transmission. Deep crimson = trapped effort.
Cosmic black = the void that responds."

[6. LUZ COMO NARRATIVA]
A luz não ilumina — ela revela. Conta parte da história.
"Divine light column from above — the cosmos recognizing the frequency."

[7. TEXTURA & ACABAMENTO]
O que dá ao espectador a sensação de que a imagem é "real" e não gerada.
"Rich painterly texture, visible brushstrokes in the cosmic elements.
Luminous particles with soft bokeh. Sacred geometry as translucent overlay."
```

### Sufixo Obrigatório:
```
"Vertical portrait, 4:5 ratio. No text, no readable symbols, no watermarks, no logos.
Multiple conceptual layers visible simultaneously.
The image is a complete visual argument — legible without any text."
```

## Fase 6 — Geração & Seleção

### Processo:
1. Aplicar Fases 1-5 para construir o prompt
2. Gerar **3 variações** com o mesmo prompt
3. Avaliar cada uma em 4 critérios:

| Critério | Pergunta | Peso |
|---|---|---|
| **Scroll-stop** | Alguém pararia de scrollar para ver isso? | 30% |
| **Conceito legível** | O argumento visual é compreensível sem texto? | 30% |
| **Profundidade** | Há camadas — algo novo aparece ao olhar mais tempo? | 20% |
| **Espaço para texto** | O terço inferior permite sobreposição sem competir? | 20% |

4. Selecionar a melhor variação
5. Se nenhuma atinge 70%+ nos critérios, refinar o prompt e gerar novamente

---

## Checklist do Diretor de Arte (pré-geração)

- [ ] Fase 1: emoção primária e secundária definidas
- [ ] Fase 2: metáfora visual sustenta 3 leituras (superficial, conceitual, filosófica)
- [ ] Fase 3: mestre(s) de referência escolhido(s) — nunca genérico
- [ ] Fase 4: composição tem tensão visual e caminho do olho definido
- [ ] Fase 5: prompt segue 7 camadas em ordem
- [ ] Cada cor tem significado narrativo (não decorativo)
- [ ] Luz como narrativa (não apenas iluminação)
- [ ] Terço inferior da imagem permite texto
- [ ] Prompt termina com sufixo de restrições

## Anti-padrões (o que NUNCA fazer)

| Anti-padrão | Por quê |
|---|---|
| "Beautiful cinematic dark image" | Genérico — Gemini gera o que quiser |
| "Energy radiating" sem especificar comportamento | Vago — não é metáfora, é efeito |
| Listar 6+ elementos sem relação entre eles | Confunde o modelo, dilui o conceito |
| Descrever atmosfera sem argumento | Decoração, não comunicação |
| "No face, no detail, pure silhouette" | Desumaniza — o espectador precisa se projetar |
| Prompt com menos de 800 caracteres | Subaproveitamento — o Gemini responde à densidade de instrução |
| Prompt com mais de 2500 caracteres | Overload — o modelo perde coerência |
