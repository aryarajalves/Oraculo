# Visual DNA — Fonte Oculta (@afonteoculta)

## Identidade Visual Real

A Fonte Oculta produz **gravuras sagradas digitais** — imagens com a densidade visual de
um Doré, a anatomia transcendental de um Alex Grey, e a força simbólica de arte esotérica clássica.

Cada imagem é um frame que para o scroll em 0,3 segundos — não por beleza abstrata,
mas por **reconhecimento visceral**: o seguidor sente que já viu aquilo dentro de si mesmo.

**Referências visuais dos carrosseis que funcionaram:**
- Gravura/engraving hyper-detalhada (Gustave Doré, William Blake)
- Anatomia sagrada e esotérica (Alex Grey)
- Chiaroscuro dramático (Caravaggio)
- Arte digital visionária com elementos simbólicos (Android Jones)
- Fotografia B&W com sobreposição de frequências/geometria sagrada

## DNA Visual — As 5 Leis Reais

### LEI 1 — Expressão Humana Próxima
- O sujeito humano ocupa **40 a 65% do frame** — close-up ou medium-close.
- **Rosto expressivo** é permitido e frequentemente o elemento principal.
- O seguidor deve se projetar no sujeito — expressão carrega o estado emocional do slide.
- Posição: levemente off-center, nunca perfeitamente centralizado.
- **NÃO**: silhueta minúscula no vazio. Isso produz distância, não identificação.

### LEI 2 — Paleta Monocromática + 1 Cor Seletiva
- **Base**: preto profundo, sépia escuro, carvão — altamente desaturado.
- **Acento**: UMA cor completamente saturada contra a base monocromática.
  - Crimson/vermelho: confronto, inflamação, sistema que falhou
  - Teal/verde-azul: frequência, transmissão, campo
  - Ouro/âmbar: revelação, portal, ascensão
  - Violeta: espiritualidade, mistério, velamento
- **NÃO**: imagem toda em cores. A força vem do contraste monocromático → saturado.

### LEI 3 — Textura de Gravura
- **Engraving/etching hyper-detalhado** em toda a imagem.
- Hachura (cross-hatching), pontilhismo (stippling), linhas finas de gravura.
- A textura é o que diferencia da arte genérica de IA.
- Background também tem textura — nunca fundo plano/liso.
- Referência: Gustave Doré's Bible illustrations em estilo digital.

### LEI 4 — Elemento Simbólico/Esotérico
- Cada imagem pode (não obrigatório) ter um elemento simbólico sobreposto:
  - Geometria sagrada (mandala, Flor da Vida, Sri Yantra)
  - Linhas de frequência/campo eletromagnético
  - Anatomia sagrada transparente (Alex Grey style)
  - Símbolo esotérico relevante ao tema
- O elemento simbólico é **secundário ao sujeito principal** — não domina.
- Renderizado em linhas finas, semi-translúcido, integrando à composição.

### LEI 5 — Chiaroscuro e Espaço para Texto
- Iluminação dramática: contraste alto, sombras profundas.
- Fonte de luz: uma só, direcional, âmbar-ouro ou específica ao tema.
- **O terço inferior do frame deve escurecer** para permitir o overlay de texto.
- A escuridão cresce de cima para baixo — o sujeito está no centro/superior,
  o texto habita o espaço escuro inferior.

## O que NUNCA aparece

| Elemento proibido | Por quê |
|---|---|
| Texto, letras, números | Sem exceção |
| Composição minimalista "vazio + silhueta" | Produz distância, não identificação |
| Paleta colorida sem foco seletivo | Dilui o impacto da cor de acento |
| Fundo liso/plano | Falta textura = parece IA genérica |
| Estética stock photo | Quebra a identidade |
| Mais de 1 cor saturada | Confunde a hierarquia visual |
| Imagem abstrata sem elemento figurativo | Não para o scroll |

## Estrutura do Prompt por Slide

### Template de prompt (7 elementos):

```
[1. ESTILO & REFERÊNCIA ARTÍSTICA]
"Hyper-detailed engraving style, Gustav Doré meets Alex Grey.
In the visual language of [artista específico para este tema]."

[2. SUJEITO PRINCIPAL]
Descrição precisa: quem/o quê, posição, expressão, escala no frame.
"Close-up 3/4 portrait of [descrição]. Figure occupies 55% of frame. Expression: [emoção]."

[3. COR SELETIVA DE ACENTO]
UMA cor. Onde aparece. Por que está lá (o que representa).
"Single accent color: deep crimson red — glowing from [localização] — representing [o quê]."

[4. ELEMENTO SIMBÓLICO (se houver)]
O símbolo esotérico. Como está integrado. Intensidade.
"Overlaid sacred anatomy: faint cardiovascular system visible through the chest, 
fine engraving lines, semi-translucent."

[5. ILUMINAÇÃO]
Fonte, ângulo, temperatura, comportamento.
"Single key light: warm amber (3000K), upper-left, 55 degrees. 
Hard edge, zero fill. Deep Caravaggio shadows."

[6. TEXTURA & BACKGROUND]
Material do fundo. Tipo de textura. Densidade.
"Background: dense cross-hatched engraving, absolute black, layered hatch marks 
giving depth to the darkness."

[7. ESPAÇO PARA TEXTO]
"Lower 30% fades to pure dark — no visual elements below midpoint."
```

## Exemplos por Arquétipo

### Arquétipo A — "Confronto Emocional"
Slides: DISRUPÇÃO, QUEDA FUNDA, NOMEAÇÃO
Referência: Caravaggio + Beksinski engraving
Cor: crimson ou ausência total de cor (puro B&W sépia)

### Arquétipo B — "Mecanismo Sagrado"
Slides: PROFUNDIDADE, ASCENSÃO
Referência: Alex Grey sacred anatomy
Cor: teal para mecanismo, ouro para revelação

### Arquétipo C — "Espelho e Reconhecimento"
Slides: ESPELHO, DESCIDA
Referência: Caravaggio intimidade + James Turrell luz
Cor: ouro suave

### Arquétipo D — "Portal e Cristalização"
Slides: CRISTALIZAÇÃO
Referência: Doré intervenção divina + Turrell threshold
Cor: ouro dourado saturado

## Checklist Pré-Geração

- [ ] Sujeito expressivo e próximo (não silhueta minúscula)
- [ ] Textura de gravura especificada
- [ ] APENAS 1 cor saturada de acento
- [ ] Cor de acento tem significado narrativo (não decorativo)
- [ ] Elemento simbólico integrado (não colado por cima)
- [ ] Terço inferior escuro para texto
- [ ] Referência artística específica nomeada
- [ ] Prompt entre 800 e 2500 caracteres

## Padrões Virais — Atualizado via Análise de 3 Carrosséis

Ver arquivo completo: `agents/canalizador-visual.md`

### Resumo dos padrões que os virais usam e nós não usávamos:

**1. ALTERNÂNCIA OBRIGATÓRIA de imagem e text_only**
Os carrosséis com mais compartilhamentos alternam entre slides visuais (emoção) e slides
de fundo preto puro (argumento lógico). O text_only cria intimidade e desaceleração deliberada.
Nunca mais de 2 imagens seguidas sem uma respiração de texto.

**2. UMA ÂNCORA CIENTÍFICA REAL por carrossel**
Foto real de: diapasão vibrando, placa de Cimática, cristal de água, scanner cerebral.
Layout: `card`. Posição: S2 ou S4. Quebra o padrão místico e cria credibilidade física.
Impacto: "não é mágica, é física" — o seguidor para de resistir.

**3. ÍCONE SAGRADO RECONHECÍVEL em S1**
Cristo (Van Gogh), Moisés (gravura Doré), Criação de Adão (Michelangelo).
Ativa confiança antes de qualquer leitura. O seguidor para porque reconhece.
Nunca usar ícone sagrado após S3 — perde peso se repetido.

**4. TEXTO NA IMAGEM: MÁXIMO 3 LINHAS**
Os virais têm 1-3 linhas de copy sobreposta à imagem. Quando precisam de mais,
mudam para text_only. Nunca comprimem copy para caber na imagem.

**5. COR DE ACENTO MUDA POR ESTADO EMOCIONAL**
Não uma paleta fixa. Violeta para portal/espírito, teal para ciência,
ouro para revelação/CTA, crimson para ruptura/nomeação.
Ver tabela completa em `canalizador-visual.md` Parte 5.
